# UnboundMechanic
This is a Terraria Mod that automatically "unties" bound NPCs in the world.
Specifically the:
- Mechanic (the reason for making the mod)
- Stylist
- Angler
- Golfer
- Goblin Tinkerer
- Tavernkeep
- Wizard
- Tax Collector

It also has functionality for enabling every NPC to spawn from the start of the game
